# Biomedical Image Analysis

## Overview

This repository documents my learning and application of computational approaches for biomedical image analysis.

My research background is in medical science and tissue regeneration, and I am interested in applying AI-enabled approaches to quantitative microscopy and translational biomedical research.

## Objectives

- Learn computational methods for microscopy image analysis
- Develop reproducible workflows for biological image quantification
- Explore AI-assisted approaches for biomedical research

- Python
- Machine learning fundamentals
